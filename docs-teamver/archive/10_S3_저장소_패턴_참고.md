## 요약

| 방식 | 용도 | Design prod |
|------|------|-------------|
| **앱 SDK wrapper** (`ProjectStorage` / boto3) | 서비스 hot path | ✅ 채택 |
| **Hybrid + scratch** | agent CWD 필요 시 | ✅ MaterializingProjectStorage |
| **rclone/s3fs mount** | 운영자 read-only 탐색 | △ incident만 |
| **s3fs (Python batch)** | design-api admin 집계 | △ cold path |

## S3에 부적합한 사용 (Design 해당)

- SQLite / Postgres data directory → **Litestream replicate**, mount 금지
- 실시간 편집·rename·lock → **scratch + sync**
- git repo, node_modules, plugin staging

## Teamver 타 앱과의 정렬

Main BE Drive·첨부파일은 **SDK 직접** (mount 아님). Design도 동일 원칙; agent scratch만 추가.
